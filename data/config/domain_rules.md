# VOC 분류 도메인 규칙

## 서비스 구분
- **어스플러스**: 커뮤니티 앱 (편지글, 게시글, 커뮤니티)
- **어스캠퍼스**: 강의 플랫폼 (VOD, 라이브, 나의스터디, 기업분석도감)
- 두 서비스는 별개. 반드시 구분하여 서술

## 기술이슈/기능요청 (개발팀 대응 = 앱/웹 기능 개발이 필요한 것)
- 앱/웹 접속 불가, 화면 오류, 버튼 미작동
- 결제 시스템 오류 (결제 실패, 미반영, 이중결제)
- 알림 미발송, 미디어 재생 불가
- 링크 오류, 콘텐츠 로딩 실패
- UI/UX 버그 (글씨 크기, 확대 기능, 스크롤 등)
- 새 앱/웹 기능 요청 (댓글, 검색, 필터 등)

## 기술이슈가 아닌 것 (운영/사업 정책 영역)

### 투자/콘텐츠 내용
- 매입단가, 차트 가격, 포트폴리오 수익률, 종목 편입/편출
- 마스터가 제공하는 콘텐츠 내용 오류/의견
- 투자 손실/수익에 대한 감정 표현

### 운영 정책
- 요금/가격/혜택 정책 의견
- 멤버십 등급/가입 조건/수강 순서 제한
- 오프라인 모임/지역 모임 개설 요청
- 커뮤니티 운영 방침 (글 삭제 기준, 차단 정책 등)
- 교재/선물 배송 관련
- **콘텐츠 업로드 시간/주기**: 브리핑 시간 변경 요청, 업로드 시간 고정 요청, 콘텐츠 게시 주기 관련 의견 등은 운영이슈 (콘텐츠 업로드는 운영팀/마스터가 수동으로 하는 것)
- **게시판/편지쓰기 개폐**: 게시판, 편지쓰기 등은 운영팀이 어드민에서 오피셜클럽별로 활성화/비활성화하는 것. "게시판 열어달라"는 운영이슈. 단, "게시판이 이상하다/오류가 난다"는 기술이슈

### 외부 서비스
- 삼성증권, 키움증권, 미래에셋 등 증권사 앱/서비스
- 유튜브, 네이버, 카카오톡 등 외부 플랫폼
- 줌 자체 문제 (줌 앱 설치, 줌 계정 등)
- 주식창/HTS/MTS 가격 표시, 호가창, 차트 등 증권사 앱 기능

### 운영진 공지
- "운영 매니저입니다", "안내드립니다", "공지사항" 등 운영진이 작성한 글
- 사용자 VOC가 아님

### CS (고객서비스)
- 환불/해지/구독 변경 요청
- 배송 문의
- 가입/인증 방법 안내 요청
