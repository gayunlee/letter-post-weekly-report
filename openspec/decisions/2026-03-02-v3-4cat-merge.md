# v3 Topic 5분류 → 4분류 전환 (콘텐츠·투자 합산)

날짜: 2026-03-02

## 배경

v3 5분류 KcBERT 파인튜닝 결과 81.9% 정확도. 벡터 KNN(83.5%), 하이브리드(84.0%)로 소폭 개선 가능하지만 여전히 부족.

**핵심 병목**: 콘텐츠 반응↔투자 담론 혼동 77건 (전체 오분류 126건의 61%)

## 분석

### 합산 시 정확도 변화 (KcBERT 기준)

| 데이터셋 | 5분류 | 4분류 (합산) | 차이 |
|----------|-------|-------------|------|
| Test (744건) | 83.1% | **93.4%** | **+10.3%p** |
| Golden (102건) | 69.6% | **77.5%** | **+7.8%p** |

합산 후 콘텐츠·투자 F1: 0.968 (support 648건)

### 벡터 KNN + 하이브리드 벤치마크 (5분류 기준)

| 분류기 | Test | Golden |
|--------|------|--------|
| KcBERT 단독 | 83.1% | 69.6% |
| 벡터 KNN 단독 | 83.5% | 63.7% |
| 하이브리드 (t=0.3) | 84.0% | 70.6% |

하이브리드로 5분류 84.0%까지 올릴 수 있지만, 4분류 합산(93.4%)에 비하면 여전히 큰 격차.

## 결정

### 1. Topic 5분류 → 4분류로 전환

| 4분류 | 설명 |
|-------|------|
| 운영 피드백 | 운영팀/사업팀이 처리할 요청·이슈 |
| 서비스 피드백 | 개발팀이 수정할 기술적 이슈·요청 |
| **콘텐츠·투자** | 마스터/콘텐츠 반응 + 시장/투자 담론 통합 |
| 기타 | 인사·잡담·테스트 |

### 2. 세부 분석은 Tag + Sentiment로 슬라이싱

5분류에서 topic으로 억지로 나누려던 구분을 tag + sentiment 조합으로 대체:

- "콘텐츠 불만" → `콘텐츠·투자` + tag:`콘텐츠 품질/깊이` + sentiment:`부정`
- "증시 여론" → `콘텐츠·투자` + tag:`시장 전망/매크로`
- "수익 불만" → `콘텐츠·투자` + tag:`수익/손실 공유` + sentiment:`부정`
- "마스터 칭찬" → `콘텐츠·투자` + tag:`마스터 소통/태도` + sentiment:`긍정`

### 3. 벡터 KNN + 하이브리드도 4분류로 리빌드

4분류 KcBERT + 4분류 벡터 KNN + 하이브리드까지 벤치마크.

## 근거

- **사용자 실제 니즈**: "콘텐츠 품질 불만" vs "증시 여론"을 구분하고 싶다 → topic이 아닌 tag 수준의 구분
- **정확도 대폭 향상**: +10.3%p (83.1% → 93.4%)는 모델 교체 없이 얻을 수 있는 가장 큰 개선
- **기존 5분류 거부 사유 재검토**: 5분류 설계 당시 "여론 통합은 분석 자유도 부족" → 그러나 tag + sentiment 슬라이싱으로 자유도를 확보할 수 있으므로 해소됨
